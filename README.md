# Проект 'AttestationWork'
 
## Описание:

Проект AttestationWork - это проект, в котором можно создавать посты и писать под ними комментарии.

## Установка:

1. Клонируйте репозиторий:
```commandline
git clone https://github.com/SergeyBordeychuk/AttestationWork.git
```
2. Установите зависимости:
```commandline
pip install -r requirements.txt
```

## Функционал
1. Запуск сервера
```commandline
python manage.py runserver
```

2. Действие на сервере
```commandline
Просмотр, создание, изменение и удаление постов, а также комментариев.
```

3. Новый функционал
```commandline
Добавлена документация по ссылке /swagger/
```

4. Закрыть сервер
CTRL+C
